About


This webgame is based on the classic game "Simon" from the 80's.
In this version we've added the possibility to make the game more complex by opting for more than four tiles!
Have fun challenging yourself!

How does it work?

  • Press the "Start New Game" button.
  • One of the tiles will now light up.
  • The white circle around the tiles indicates it's your turn: repeat the pattern that was demonstrated.
  • If you've repeated the pattern correctly, the game will now repeat the pattern and add a new tile to the pattern at the end.
  • Repeat!
Try a game with more tiles if it isn't challenging enough (or find the secret challenge). Have fun!
